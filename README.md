# 📈 Matplotlib Data Visualization

## 📌 Overview
This repository documents my learning journey with **Matplotlib**, one of the most widely used Python libraries for data visualization. It contains examples of various charts and plots used for analyzing and presenting data effectively.

## 🎯 Objectives
- Learn the fundamentals of data visualization
- Create different types of plots using Matplotlib
- Understand chart customization and styling
- Perform basic Exploratory Data Analysis (EDA)

## 📚 Topics Covered
- Line Plot
- Bar Chart
- Scatter Plot
- Histogram
- Pie Chart
- Box Plot
- Subplots
- Plot Customization
- Legends and Annotations
- Grid and Styling

## 🛠️ Technologies Used
- Python
- Matplotlib
- NumPy
- Pandas
- Jupyter Notebook
